Programación
Orientada a Objetos

Cada objeto es una caja que tiene cosas dentro — datos y acciones organizados juntos.

01 — CONCEPTO BASE
📦

Clase / Objeto

La clase es el molde. El objeto es lo que se construye con ese molde.

Molde = Clase Resultado = Objeto
02 — DATOS
🚗

Atributos

Son las características del objeto. Describen cómo es.

Color = Rojo Marca = Toyota Velocidad = 0
03 — ACCIONES
⚙️

Métodos

Son las funciones del objeto. Lo que puede hacer.

acelerar() frenar()
04 — SEGURIDAD
🔒

Encapsulación

Oculta los datos para que no se modifiquen directamente. Se accede a ellos solo mediante métodos.

velocidad = -100 ❌ frenar() ✅
05 — CREAR OBJETO
🏭

Instanciar

Instanciar = tomar la clase (molde) y crear un objeto con sus propios valores.

const miCarro = new Carro("Rojo", "Toyota");
06 — INICIALIZAR
🔧

Constructor

Método especial que se ejecuta al crear el objeto. Le da los valores iniciales.

constructor(color, marca) {
  this.color = color;
  this.marca = marca;
}
07 — ORGANIZACIÓN
🧩

Modularidad

Dividir el programa en partes pequeñas. Cada parte tiene una función específica.

carro.js motor.js ruedas.js
08 — EJEMPLO COMPLETO
💻

Todo junto en código

class Carro {
  constructor(color, marca) {
    this.color = color;
    this.marca = marca;
    this.#velocidad = 0;
  }

  acelerar() { this.#velocidad += 10; }
  frenar()   { this.#velocidad -= 10; }
  info()     { console.log(`${this.marca} → ${this.#velocidad} km/h`); }
}

const miCarro = new Carro("Rojo", "Toyota");
miCarro.acelerar();
miCarro.info(); // Toyota → 10 km/h
